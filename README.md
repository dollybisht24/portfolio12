# Portfolio (Vite + React)

Minimal scaffold for a portfolio using Vite and React.

Quick start:

```bash
cd portfolio
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

Files added:
- package.json
- vite.config.js
- index.html
