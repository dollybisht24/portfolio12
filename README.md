# Dolly Bisht - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience.

## Features

- **Responsive Design**: Works seamlessly on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, attractive design with smooth animations and transitions
- **Interactive Elements**: 
  - Typing effect for job titles
  - Smooth scrolling navigation
  - Animated skill bars
  - Fade-in animations on scroll
  - Interactive hover effects
- **Sections**:
  - Hero section with animated profile
  - About me with education and contact info
  - Technical and soft skills
  - Featured projects
  - Certifications and awards
  - Hobbies and interests
  - Contact form

## Technologies Used

- HTML5
- CSS3 (with custom properties and animations)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Poppins)

## How to Use

1. Simply open `index.html` in your web browser
2. Or deploy to any static hosting service:
   - GitHub Pages
   - Netlify
   - Vercel
   - Any web server

## Customization

To customize this portfolio for yourself:

1. Update personal information in `index.html`
2. Modify color scheme in CSS variables (`:root` section in `style.css`)
3. Add your own project links
4. Update social media links
5. Replace placeholder content with your own

## Color Scheme
